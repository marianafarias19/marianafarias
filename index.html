Projetos

Pet Life

O Petshop Petlife é uma aplicação web desenvolvida para simular um site comercial de um pet shop. tecnologias utilizadas foram: HTML, CSS e JS.


DBurguer

O site “Cardápio Online D’Burguer” é uma aplicação web responsiva que permite aos usuários explorar um cardápio de forma prática e intuitiva. tecnologias utilizadas foram: HTML, Tailwind CSS, JS e Next JS.


Reserva de Hotel

O site simula uma aplicação para fazer reservas de hotel, onde é possível selecionar datas de entrada e saída, alem de poder escolher o tipo de quarto. Tecnologias utilizadas foram: HTML, CSS e JS.
